Perbandingan

Kenapa Guna PC Lama?

PC Terpakai

  • Kos permulaan hampir percuma
  • Mengurangkan sisa elektronik (Kitar Semula)
  • Bebas pilih Sistem Operasi
  • Proses pembelajaran teknikal yang tinggi
  • Boleh di-upgrade pada masa depan

NAS / Cloud

  • Kos peranti NAS sangat mahal
  • Langganan Cloud (bayaran bulanan berterusan)
  • Terikat dengan ekosistem jenama (Vendor Lock-in)
  • Data privasi di tangan pihak ketiga

Modul 01

Asas Ubuntu 26.04 LTS & Terminal

Linux adalah tulang belakang kepada majoriti server di dunia. Kenapa Ubuntu 26.04 LTS? Kerana ia popular, percuma, stabil, dan sokongan komuniti yang sangat meluas.

Kita akan belajar berinteraksi dengan server tanpa mouse, hanya menggunakan Command Line Interface (CLI).

  • Login via SSH
  • Navigasi File System (/etc, /var, /home)
  • Update OS (apt update & upgrade)
ssh user@homelab
user@homelab:~$ sudo apt update
[sudo] password for user:
Hit:1 http://archive.ubuntu.com/ubuntu resolute InRelease
Get:2 http://security.ubuntu.com/ubuntu resolute-security InRelease
Reading package lists... Done

user@homelab:~$ mkdir server_baru

Modul 02

Kuasa Docker Containers

Docker membolehkan kita jalankan aplikasi dalam "kotak" terasing. Bersih, laju, tiada konflik software.

Tanpa Docker

Satu OS, pelbagai software bercampur aduk. Jika satu software rosak atau perlukan versi PHP berbeza, seluruh server mungkin terjejas (dependency hell).

Dengan Docker

Setiap aplikasi hidup dalam 'Container' sendiri. Lengkap dengan persekitaran yang diperlukan. Boleh dipadam dan dibina semula dalam beberapa saat.

Command Asas

docker ps
docker images
docker stop <id>
docker rm <id>
Modul 03

CasaOS & Web Apps

Bosan dengan skrin hitam terminal? CasaOS membawakan dashboard visual yang elegan untuk menguruskan server dan menginstall aplikasi dengan satu klik.

curl -fsSL https://get.casaos.io | sudo bash

Langkah Pemasangan & Akses CasaOS

01

Salin & Jalankan

Salin kod di atas, tampal (*paste*) ke dalam terminal SSH Ubuntu anda dan tekan Enter.

02

Tunggu Pemasangan

Sistem akan memuat turun dan memasang semua fail yang diperlukan secara automatik (2-5 minit).

03

Dapatkan IP Akses

Apabila siap, terminal akan memaparkan alamat IP peranti anda (cth: http://192.168.x.x).

04

Buka Web & Daftar

Buka IP tersebut di pelayar web, kemudian daftar akaun pentadbir (*admin account*) baru anda.

File Browser

Urus fail server anda terus melalui pelayar web. Upload, download dan kongsi seperti Google Drive.

Jellyfin

Bina "Netflix" anda sendiri. Stream koleksi filem dan siri TV peribadi ke Smart TV atau telefon.

Uptime Kuma

Dashboard monitoring untuk memantau status server dan rangkaian anda. Terima notifikasi jika 'down'.

Vaultwarden

Password manager peribadi (Bitwarden self-hosted). Simpan kata laluan dengan selamat dalam server sendiri.

Pi-Hole

Penyekat iklan peringkat DNS. Menapis trackers & iklan sebelum ia sampai ke peranti anda.

MySpeed

Memantau & merekod kelajuan internet anda secara berkala secara automatik dalam graf interaktif.

Tailscale VPN

Membina rangkaian VPN peribadi yang selamat untuk mengakses server anda dari luar rumah.

Hermes AI

AI Agent pintar hos sendiri dengan memori berterusan yang boleh dihubungkan ke Telegram/Discord.

Cabaran

Checklist Troubleshooting

  • Tak boleh akses IP? -> Semak Router/Firewall
  • Docker gagal run? -> Check 'docker logs'
  • Port conflict? -> Tukar port dalam setting
  • ✓ Golden Rule: Cuba Reboot Dahulu!
